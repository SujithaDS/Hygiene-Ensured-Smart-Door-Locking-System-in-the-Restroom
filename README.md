# Hygiene-Ensured-Smart-Door-Locking-System-in-the-Restroom

An Arduino-based smart restroom automation system that enhances **hygiene, privacy, and safety** through touch-free door control and sensor-based monitoring.

## Features

* Touch-free door operation
* Occupancy detection using IR sensors
* Smart door locking for privacy
* Door opens after flush detection
* Emergency alert with buzzer
* Real-time status display on 16×2 LCD

## Tech Stack

* Arduino Uno
* Embedded C
* Arduino IDE
* IR Sensors
* Servo Motor
* 16×2 LCD
* Buzzer

## Workflow

1. User approaches the restroom.
2. IR sensor detects the user and opens the door.
3. Door closes and locks after entry.
4. Flush is detected using an IR sensor.
5. Door unlocks and opens after flush detection.
6. Emergency button triggers the buzzer when pressed.
